## คำอธิบายโปรเจค:
Shoppy เป็นเว็บไซต์ร้านค้าออนไลน์ (E-Commerce) ที่ออกแบบมาเพื่อให้ผู้ใช้สามารถเลือกซื้อสินค้าเทคโนโลยี เครื่องใช้ไฟฟ้า อุปกรณ์เสริม และสินค้าไลฟ์สไตล์ได้อย่างสะดวก รวดเร็ว และปลอดภัย เว็บไซต์ประกอบด้วยหน้า “หน้าแรก”, “เกี่ยวกับ”, “บริการ”, “แพ็กเกจราคาเปรียบเทียบ”, และ “ติดต่อเรา” พร้อมระบบเมนูนำทางที่ใช้งานง่าย เพื่อมอบประสบการณ์การช้อปปิ้งที่ครบวงจรแก่ผู้ใช้งาน
